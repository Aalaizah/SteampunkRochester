I've also started working on backgrounds for scenes, mainly the Editor's Office. The biggest issue I faced with these designs is realizing NOW, post character drawing/coloring, that it's hard to design and draw backgrounds for each scene because each character already has an angle and body position. And if there are more than one or two characters in a scene (like the Editor's Office which should have 4) it becomes incredibly difficult because I have to design the scene (a very small room) so that all of the characters and their angles appear correct. I also want to talk to coding team about the screen panning in such a small room.
