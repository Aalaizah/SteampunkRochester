Steampunk Rochester Ramp Up

19 March 2015

The Beginnning

Production studio has taken a bit longer than most of my other project based courses to really start ramping up. It's definitely nice to have had that breather at the beginning of the semester to get everything else started off right. Before we even broke into teams of who's going to do what we had to start getting a concrete plan of what the game would look like. We had to familiarize ourselves with the content that was given to us, and the time period it needed to take place in before we could decide on even a genre of game.

Production Fun

After we finally got the game off the ground the next step for me was to set up this blog. It's still sort of a work in progress, it might need to be migrated to a new server that can handle the load a little better at some point, but for now things look like they're working. Each of the teams seem to be finally getting things set in stone and we have some very basic functionality in a grey box done. It should be interesting to see where things go from here, but it looks like it will be a really cool game.

Alexandria Mack
