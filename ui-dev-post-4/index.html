UI Dev Post 4

20 April 2015

Interviews, a couple of those popped up and for the past few days I have been working on those. But I was able to enjoy some BrickHack culture this weekend while working on setting up the new wireframe designs. My goal for this new wireframes are to better show what we want in our final designs.

One idea AJ and I were playing around with is placing the map on an angle so we can have a skybox in the background to help tell the time of the day. This would be better than just having a time of the day UI element floating the top left corner. This would be more dynamic. To test this idea, I took a map and imported it into Unity. I placed it on an empty object with a sprite renderer. We then placed 3D objects to simulate pins on the map. These objects in combination with Unity lighting makes some pretty cool shadows that we could use to also indicate the time of day.

Another idea we threw around was having dynamic text boxes on the screen to help liven up the dialogue screen. We wanted to stray away from the static text screen.

I plan to have another set of designs ready for review by next class.

Joe Coppola
